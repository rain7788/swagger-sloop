# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-29

### Added

- Initial release
- Beautiful Swagger UI skin inspired by Art-Design-Pro
- Dark/Light theme support with auto-detection
- Global search feature (Ctrl+K)
- Code copy functionality (cURL, JavaScript, C#/Java)
- Multiple API groups support
- Responsive design for mobile/tablet/desktop
- .NET support (6.0, 7.0, 8.0, 9.0)
- Java Spring Boot support (2.7+, 3.x)
- NuGet package for .NET
- Maven artifact for Java
- Customizable primary color
- Additional CSS/JS injection
- OAuth2 configuration support
